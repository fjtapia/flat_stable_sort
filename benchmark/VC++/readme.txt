For to run the benchmarks compile the program file_generator.cpp.

After this run this program and create the random data file input.bin
All the benchmark programs use the same input.bin file

The command is 

file_generator input.bin 125000000

After this compile the benchm ark program and execute. The results appear on screen.

When finish delete 

    * file_generator , 
    * the benchmark program
    * input.bin
